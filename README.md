# Gaming Website

Welcome to the Gaming Website! This project provides a fun and interactive platform for gamers to explore, learn how to play, and connect.

## Features
- **Dynamic Hero Section**: A visually appealing welcome section with navigation buttons.
- **How to Play **: Step-by-step instructions for new players.
- **Gallery**: A dedicated page showcasing game images and media (linked through a button).
- **Contact Section**: Easy access to connect via Telegram, email.


## Technologies Used
- **HTML**: Structure of the website.
- **CSS**: Styling and animations.
